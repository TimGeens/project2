Cursussen

In 2020 heb ik de ‘Complete C# Unity Game Developer 3D Online Course’ aangekocht op Udemy. In de cursus werd je aangeleerd hoe je met Unity kan werken en je kreeg er ook een goede basis in C#. De cursus bestond uit het maken van verschillende kleine videogames, zoals een tower defence game, een zombie first person shooter en een rail shooter. Van dezelfde makers heb ik ook een Blender cursus en een Pixel Art cursus gekocht en gevolgd.

Unity game engine

Programmeren deed ik tot nu toe vooral in de Unity game engine. De cursussen die ik gevolgd heb hadden vooral met doel om beter met de Unity te kunnen werken. Mijn Game Jam game, Ascending Acension, is ook gemaakt in deze engine.
